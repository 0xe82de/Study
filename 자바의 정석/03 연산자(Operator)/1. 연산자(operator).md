| 단항 연산자 |    <-    | ++, --, +, -, ~, !, (type)                    |   높음   |

| 산술 연산자 |    ->    | +, -                                          |          |
| 산술 연산자 |    ->    | <<, >>                                        |          |

| 비교 연산자 |    ->    | ==, !=                                        |          |
| 논리 연산자 |    ->    | &                                             |          |
| 논리 연산자 |    ->    | ^                                             |          |
| 논리 연산자 |    ->    | \|                                            |          |
| 논리 연산자 |    ->    | &&                                            |          |
