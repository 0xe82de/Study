# 자바의 정석

"자바의 정석" 책을 정리하였습니다.

- 제가 정리한 내용은 "자바의 정석" 책의 모든 내용을 포함하지 않습니다.
- 전체 내용 중 제가 다시 볼만한 내용을 위주로 정리하였습니다.
- 정리한 내용 중 정확하지 않거나, 틀린 부분이 있을 수 있습니다.

**따라서, 처음 자바를 공부하시는 분들은 자바의 정석 책을 보시는 것을 추천합니다.**
